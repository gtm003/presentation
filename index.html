What is SVG?

  • SVG stands for Scalable Vector Graphics
  • SVG is used to define vector-based graphics for the Web
  • SVG defines the graphics in XML format
  • Every element and every attribute in SVG files can be animated
  • SVG is a W3C recommendation
  • SVG integrates with other W3C standards such as the DOM and XSL

The SVG `path` Syntax

CSS3 animations

SVGs can be animated by adding an id or a class to the SVG path in the code and then styling it in CSS3 like any other document.

Below is an example of how SVGs can be animated.

CSS


                            #circle {
                                animation: color 10s infinite alternate;
                            }
                            @keyframes color {
                                0%   {
                                    fill:  yellow; 
                                }
                                100% {
                                    fill: red; 
                                }
                            }
                            
                            #rect {
                                animation: size 10s infinite alternate;
                            }
                            @keyframes size {
                                0%   {
                                    width:  0;
                                height: 0;
                                transform: translate(55px, 55px);
                                }
                                100% {
                                    width:  110;
                                height: 110;
                                transform: translate(0px, 0px);
                                }
                            }
                            
                            #clock text {
                                font-size: 16px;
                                font-family:sans-serif;
                                fill:black;
                                stroke:none;
                            }
                              
                            #arrow {
                                  fill: black;
                                stroke:none;
                                animation: rotate 60s infinite linear;
                            }
                              
                            @keyframes rotate {
                                  0% { 
                                    transform: rotate(0turn);
                                    transform-origin: 53px 53px;
                                }
                                  100% { 
                                    transform: rotate(1turn);
                                    transform-origin: 53px 53px;
                                }
                            }

The animate tag animations

    Specifying the target of the animation:
  1. Using the xlink:href attribute
  2. If the xlink:href attribute is not provided, then the target element will be the immediate parent element

The animate tag animations

    The important attributes for <animate> are:
  • attributeName  -  The name of the attribute to animate.
  • from  -  The initial value of the attribute.
  • to  -  The final value.
  • dur  -  The duration of the animation (for example, write '5s' for 5 seconds).
